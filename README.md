Example of how to consume the api Firefox OS contacts.

- List All Contacts
	
  	navigator.mozContacts.getAll();

- Find Contact
	
	navigator.mozContacts.find(params);

- Save Contact

	navigator.mozContacts.save(contact);

- Remove Contact

	navigator.mozContacts.remove(contact);
